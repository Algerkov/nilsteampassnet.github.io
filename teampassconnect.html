TeampassConnect

TeamPassConnect is a web browser extension permitting to close relate Items in Teampass database and the websites browsed.

 

Now in Beta for Firefox

TeampassConnect extension is now available for Firefox. Install it and request your first month trial to discover its useful features. Using TeampassConnect allows you to interact in a bidirectionnal way between Teampass database and the web borwser.

 

Download and install

Download the Firefox XPI file and start using TeampassConnect.
